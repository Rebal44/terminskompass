# Terminskompass

Public sales page and limited interactive demo for Terminskompass, a Swedish
offline study planner.

The paid application source and buyer download are not included in this
repository.

## Privacy

The demo does not use accounts, analytics, cookies, or remote data storage.

## Product

Terminskompass helps students organize deadlines, generate an exam study plan,
and run focused study sessions. The paid product is delivered as a small offline
web app for personal use.
